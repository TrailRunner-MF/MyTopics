カスタムプロジェクトテンプレートを1度削除すると2度とCreate a new project Dialogに表示されない
--------

簡単なVisual StudioのC#のプロジェクトテンプレートを作ってみました。
参考にしたのは以下のようなページです。

https://docs.microsoft.com/ja-jp/visualstudio/ide/creating-project-and-item-templates?view=vs-2019
https://docs.microsoft.com/ja-jp/visualstudio/ide/how-to-create-project-templates?view=vs-2019

この処理は上手くいき、Create a new project Dialog Window上ではプロジェクト名で検索可能で、選択するとちゃんとテンプレートを元にしてプロジェクトとソリューションを作ってくれます。

このテンプレートの名前はSimpleTemplateです。テンプレートのエクスポート処理により以下のファイルが出力されたことが確認できました。
C:\Users\[UserName]\Documents\Visual Studio 2019\Templates\ProjectTemplates\SimpleTemplate.zip

この手順を再確認するために、いったんこのzipファイルを削除してもう一度この手順を繰り返してました。即ち同名のプロジェクトテンプレートをもう一度最初から作り直し、Visual Studioのメニューよりエクスポートします。

するとどういう訳かそれ以降、Visual Studioは一切自作のプロジェクトテンプレートを読み込めなくなります。
単に作り直した同名のプロジェクトテンプレートだけでなく、SimpleTempate2とかSimpleTemplate3を作成しても、Create a new project Dialog Window上では検索できません。キーワードを入れてもタグで選択しようとしても一切表示されません。

興味のある方は是非試してみてください。これは自分の環境においてのみ発生しているのか？あるは他の人の環境でも発生するのか？

自分の環境は以下の通りです。
プロセッサ：Intel(R) Core(TM) i5-24005 CPU @2.50GHz
実装メモリ 16.0GM
システムの種類： 64ビットオペレーションシステム
Windowsのエディション：Windows 10 Enterprise
Microsoft Visual Studio Professional 2019 Version 16.5.2

この問題を解決するための情報をお持ちの方は、是非お伝えください。よろしくお願いします。

--------

https://docs.microsoft.com/ja-jp/visualstudio/ide/creating-project-and-item-templates?view=vs-2019
https://docs.microsoft.com/ja-jp/visualstudio/ide/how-to-create-project-templates?view=vs-2019

This works well, and in the Create a new project Dialog Window, you can search by project name, and when selected, it will create a project and solution based on the template.

The name of this template is 'SimpleTemplate'. I confirmed that the following files were generated by the template export process.
C:\Users\[UserName]\Documents\Visual Studio 2019\Templates\ProjectTemplates\SimpleTemplate.zip

I deleted the zip file and repeated the process again to reconfirm this step.
To be specific I deleted the project once, and recreated from begning, and then exported exactly the same way I did it in a few menutes before.
